# Vue 项目的搭建

## 一、步骤

1. 安装nodejs

   网址：http://nodejs.cn/download/Vue  （CLI 3需要 nodeJs ≥ 8.9 ）

2. 安装@vue/cli（Vue CLI 3的包名称由 `vue-cli` 改成了 `@vue/cli`）

   ```
   cnpm install -g @vue/cli 
   // yarn global add @vue/cli
   ```

3. **搭建新项目**

   ```
   vue create <Project Name> ``//文件名 不支持驼峰（含大写字母）
   ```
